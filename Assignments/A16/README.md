## Sorting Experiment
### Taylor Mendez 
### Description:

Creating an experiment to compare the sorting routines we
discussed in class. I will vary the following about the 
data being sorted:
- size
- type: integers, floats, strings
- discombulation of data:
    - completely sorted or random
    - partially unsorted
    - reverse sorted
    - already sorted
